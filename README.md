# SYNAPSE FEST – Festival Management Database System

A PostgreSQL-based database system designed for managing a college techno-cultural festival. The project models various aspects of festival operations such as events, committees, attendees, tickets, sponsors, accommodation, vendors, and lost-and-found management.

## Features

* Designed and normalized a relational database schema up to BCNF.
* Implemented 30+ entities with appropriate primary keys, foreign keys, and integrity constraints on PostgreSQL.
* Developed PostgreSQL triggers for accommodation billing, room-capacity validation and handling event-venue confliction.
* Created realistic sample datasets and analytical SQL queries for ticket sales, sponsorships, committee funding, and accommodation management.

## Tech Stack

* PostgreSQL
* SQL
* pgAdmin
